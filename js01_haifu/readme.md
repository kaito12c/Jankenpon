じゃんけんに負けて、脳年齢を鍛える。

## ②工夫した点・こだわって点
１：世界観。
２：おばあちゃんが好みそうな「ダ鳥獣戯画」
　　勝つのではなく、負ける仕組み。

３：30秒以内に何問正解できたか、で
　　脳年齢が計測できる

　　正解のカウントし、結果に
　　何問中何問正解かを出すのが難しかった。
